# Proyecto-Integrador-U2
Explicación del fotograma que se realizó en Blender 


---

## Objetivo del proyecto
El objetivo fue desarrollar una animación fluida de un perro, lobo o cualquier animal u objeto (en este caso se eligió un perro café) basada en una hoja de poses (sprite sheet). 


---

## Herramientas utilizadas
* **IA Generativa:** Google gemini y ChatGPT
* **Software de Animación:** Blender y FlipArtify
  

---

## Proceso de Elaboración 

### 1. Generación de la Referencia con IA
Se utilizó un prompt específico para obtener una **hoja de modelo (Model Sheet)**. La IA permitió definir:

  * **Consistencia Visual:** Mantener proporciones, colores y rasgos (orejas, cola, manchas) en diferentes ángulos.
  * **Secuencia de Movimiento:** Obtener las fases de contacto y elevación de las patas.


### 2. Configuración del Proyecto 
  * Se importó la imagen de referencia en una capa base con **30% de opacidad**.
  * Se configuró el lienzo a **12 FPS** (fotogramas por segundo) para un control preciso del movimiento.
  * Se activó el **Papel Cebolla (Onion Skinning)** para visualizar el dibujo anterior y posterior, garantizando que el volumen del perro no cambiara drásticamente entre cuadros.


### 3. Creación de Poses Clave (Keyframes)
Siguiendo la imagen de referencia, se dibujaron primero las 4 poses principales del ciclo:
1.  **Contacto:** Las patas están en su máxima extensión.
2.  **Paso Bajo (Down):** El cuerpo baja ligeramente al recibir el peso.
3.  **Paso Medio (Passing):** Una pata cruza a la otra.
4.  **Paso Alto (Up):** El cuerpo alcanza su punto máximo de altura antes del siguiente paso.


### 4. Refinamiento y Arcos
* **Movimiento de la Cola:** Se animó siguiendo una trayectoria en "ocho" o arcos suaves para evitar rigidez.
* **Inbetweens:** Se crearon dibujos intermedios para suavizar la transición entre las poses clave de la referencia.


### 5. Limpieza, Color y Loop
* **Lineart:** Se trazó la línea final sobre los bocetos.
* **Color Flat:** Se aplicó la paleta de colores extraída de la imagen de la IA.
* **Sincronización del Loop:** Se aseguró que el último frame conectara perfectamente con el primero, eliminando cualquier "salto" visual en la repetición.
